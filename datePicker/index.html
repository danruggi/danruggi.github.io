Easy usage:

1. Include CSS and JS

<link rel='stylesheet' href='css/deskyCal.css' onload="this.media='all'">
<script type="text/javascript" src="js/deskyCal.js"></script>

2. Create any field with an unique ID in the page

<input id='unique_id'>

3. Init the calendar with

<script>
 initDeskyCal(ID);
</script>

Example 1: Simple two columns

Two column is default
Just call the function passing just the id

// Just one parameter
initDeskyCal("calendar_date_selector1");
// Or Passing the "double" it's the same
initDeskyCal("calendar_date_selector1", "double");

Example 2: One Column

Two column is default
Just call the function without parameters

initDeskyCalendar("calendar_date_selector2", "single");
